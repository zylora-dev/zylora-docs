## How You're Billed

Zylora charges for GPU compute time in **milliseconds** — from the moment your function starts executing to the moment it returns. You never pay for idle time.

**Cost = (GPU rate per second) × (execution time in ms) / 1000**

All billing is in integer cents internally. Displayed as dollars in the dashboard.

## Credits

- Credits are pre-purchased and deducted per invocation
- Unused credits roll over month-to-month (no expiry)
- Auto-recharge available when balance drops below a threshold
- [Startup program](/docs/concepts/pricing) offers $10,000 in credits

## Spend Controls

Set a monthly or per-function spend cap via the Dashboard or API:

```bash
# Set a $500/month global cap
zylora config set spend-cap 50000  # integer cents
```

When the cap is reached, invocations return a `QuotaExceededError` until the next billing period or until the cap is increased.
