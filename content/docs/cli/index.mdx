## Installation

```bash
cargo install --git https://github.com/zylora-dev/zylora-cli --bin zy
```

The CLI command is `zy`.

## Global Flags

| Flag | Description |
|---|---|
| `--api-key <key>` | Override the API key for this command |
| `--org <slug>` | Override the active organization |
| `--json` | Output as JSON (machine-readable) |
| `--help` | Show help |

## Auth

### `zy login`

Open browser and authenticate via OAuth.

```bash
zy login
```

### `zy logout`

Remove stored credentials.

```bash
zy logout
```

## Functions

### `zy deploy`

Build and deploy a function.

```bash
zy deploy <file.py> [options]

Options:
  --name <slug>         Override function name (default: file basename)
  --tag <tag>           Tag this version
  --dockerfile <path>   Path to custom Dockerfile
  --no-wait             Return immediately without waiting for build
  --json                Output deploy result as JSON
```

### `zy invoke`

Invoke a deployed function.

```bash
zy invoke <function> [options]

Options:
  --input <json>        Input JSON string
  --input-file <path>   Read input from file
  --version <v>         Pin to a specific version
  --stream              Stream output (for streaming functions)
  --json                Output result as JSON
```

### `zy list`

List functions in the active org.

```bash
zy list [--json]
```

### `zy delete`

Delete a function and all its versions.

```bash
zy delete <function> [--yes]
```

## Versions

### `zy versions`

List versions of a function.

```bash
zy versions <function>
```

### `zy rollback`

Roll back to a previous version.

```bash
zy rollback <function> --version <v>
```

### `zy traffic`

Split traffic between versions.

```bash
zy traffic set <function> --version <v1> --weight 80 --version <v2> --weight 20
```

---

## Logs

### `zy logs`

Stream invocation logs.

```bash
zy logs <function> [options]

Options:
  --tail            Follow live logs
  --since <time>    Show logs since (e.g., 1h, 30m, 2026-04-01)
  --limit <n>       Number of log lines (default: 100)
```
